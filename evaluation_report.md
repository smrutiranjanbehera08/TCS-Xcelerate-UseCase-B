### TCS Xcelerate Program – Use Case B

---

# 📌 Project Overview

The Multi-Agent Financial Research Analyst is an AI-powered investment research system designed to automate the equity research workflow.

The project utilizes a multi-agent architecture where specialized agents collaborate to collect financial information, retrieve company news, analyze business profiles, compare industry peers, and generate structured investment thesis reports.

The system was further enhanced through Google Agent Development Kit (ADK) integration, enabling modular tool orchestration and session-based memory management.

---

# ✅ Key Achievements

The original implementation used fixed peer companies for all evaluations.

The system was enhanced with dynamic sector-based peer selection.

Examples:

| Company   | Peer Companies                     |
| --------- | ---------------------------------- |

| TCS       | Infosys, Wipro                     |

# ⚠️ Areas for Future Improvement

Although the system performs well, several enhancements can further improve performance:

### Advanced AI Thesis Generation

Current thesis generation uses structured templates.

Future versions can leverage:

* Google Gemini
* Groq LLM
* Large Language Models (LLMs)

to generate more detailed investment insights.

### Enhanced News Filtering

Additional filtering techniques can improve relevance and reduce unrelated news articles.

### Cloud Deployment

Deployment on Google Cloud Run or Streamlit Cloud can improve accessibility and scalability.

### Persistent Agent Memory

Long-term memory can be added for maintaining historical company analysis and user sessions.

# 🏁 Overall Conclusion

The Multi-Agent Financial Research Analyst successfully achieved its primary objective of automating the investment research workflow through a collaborative multi-agent architecture.

The system demonstrated reliable performance across ten publicly listed Indian companies, successfully retrieving financial information, company profiles, industry news, peer comparisons, and generating structured investment thesis reports.

The introduction of Dynamic Peer Selection and Google ADK Integration significantly improved the system architecture, making the project more scalable, modular, and aligned with modern AI agent development practices.

Overall, the project demonstrates a strong implementation of agent-based financial research automation and satisfies the key requirements of TCS Xcelerate Program – Use Case B.
